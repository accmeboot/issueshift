# Issueshift
Agile board web app

## Web app with go and htmx

## Development 

### Start dev server
```console
make run/dev
```

### Create migrations
```console
make migrations/create
```

### Run migrations
```console
make migrations/up
```
```console
make migrations/down
```

- https://github.com/go-chi/chi
